# Changelog

## [0.4.0][] - 2024-06-04

- New-type enum support.
- Container attribute `#[dynamodel(tag = "...")]` for named struct.

## [0.3.1][] - 2024-05-27

- Make the enum `ConvertError` Send and Sync.

## [0.3.0][] - 2024-05-26

- Enum support
- Generics support
- Field attributes `try_from_item` and `skip_into`.

## [0.2.0][] - 2024-05-19

- Rename attribute from `table_key` to `extra`

## [0.1.0][] - 2024-05-19

- pre-release
